Jet Onay

Mobile Digital Signature solution

  • Native Android & iOS apps enable the user to digitally sign their documents using their GSM network created certificates.
  • Fully secure as only the hash of the file is leaving the mobile device for signing. The whole CMS (PKCS7) file is generated on the device.
